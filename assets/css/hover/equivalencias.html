⚠️ Nota sobre justify-items en Bootstrap

Bootstrap no tiene clases nativas para justify-items. Para un control exacto, necesitarás CSS personalizado o usar justify-content como alternativa.

✅ Recomendaciones de Migración

🎯 Migración Estratégica

1. Mantener clases personalizadas para:

  • Las clases .inline-* (grid-auto-flow: column no existe en Bootstrap)
  • Las clases .content-1 y .content-4 (valores específicos)
  • Cualquier justify-items (no soportado nativamente)

2. Migrar a Bootstrap para:

  • Flexbox básico (.d-flex, .flex-wrap)
  • Espaciado con gap (usar escala Bootstrap)
  • Alineación justify-content y align-items
  • Anchos con .w-100

3. Crear utilidades híbridas:

/* Ejemplo de clase híbrida */
.inline-center-center {
  /* Usar Bootstrap donde sea posible */
  @extend .d-grid;
  @extend .align-items-center;
  @extend .justify-content-center;
  /* Añadir lo que Bootstrap no soporta */
  grid-auto-flow: column;
}
🚀 Próximos Pasos

Para el proyecto de Mercería Larraz, te recomiendo:

  1. Auditar qué clases personalizadas se usan actualmente en el proyecto
  2. Priorizar la migración de las clases más comunes a Bootstrap nativo
  3. Mantener un archivo de utilidades custom solo para lo que Bootstrap no cubre
  4. Documentar las decisiones de diseño para mantener consistencia
